¿Cómo se instala?

Setup rápido y configuración inicial

1

Inicializar el proyecto

npx shadcn-ui@latest init

Este comando configura todo lo necesario en tu proyecto existente

2

Seleccionar configuración

Framework
  • Next.js
  • Vite
  • Remix
  • Gatsby
Tailwind Config
  • CSS variables
  • Base styles
  • Plugin config
Alias y Paths
  • @/components
  • @/lib/utils
  • Import paths
Componentes
  • Default styles
  • Color scheme
  • Border radius
3

Agregar componentes

# Agregar componentes individuales
npx shadcn-ui@latest add button
npx shadcn-ui@latest add input
npx shadcn-ui@latest add dialog

# O múltiples a la vez
npx shadcn-ui@latest add button input dialog card
4

¡Listo para usar!

import { Button } from "@/components/ui/button"
import { Input } from "@/components/ui/input"

export function MyComponent() {
  return (
    

      
      
    

  )
}
